# Pokédex JS

In this project I use pure Javascript to replicate a Pokédex from the famous franchise Pokémon!

To achive my goal two really cool Pokemon related APIs were used:

+ PokéAPI - https://pokeapi.co/
+ Pokémon (Pokeres) API - https://pokeres.bastionbot.org
